# 🛡️ Log Analyzer — AI SOC Log Analyzer

🌐 Live Demo:https://log-analyzer-jsxc.onrender.com

An AI-powered security log analyzer built for SOC operations. Paste system logs and get instant threat detection, alert generation and AI-driven recommendations.

## 🚀 Features
- 🔍 Detects suspicious activity in system logs
- 🚨 Real-time alert generation
- 🤖 AI threat assessment using Groq LLM
- 📊 Splunk-style professional dashboard
- 🛡️ Supports SSH, Web, Firewall and Auth logs

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq API (Llama 3.1)
- **Frontend:** HTML, CSS, JavaScript

## ⚙️ Installation
```bash
# Clone the repo
git clone https://github.com/isaifmohammed/log-analyzer.git

# Install dependencies
pip install -r requirements.txt

# Add API key to .env file
GROQ_API_KEY=your_groq_key

# Run the app
python app.py
```

Open browser at http://127.0.0.1:5000
